## Tabs

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+Shift+T` | Duplicate tab |
| `Cmd+Shift+R` | Reload tab |
| `Cmd+W` | Close tab |
| `Cmd+1`–`9` | Switch to tab |
| `Cmd+Shift+[` | Previous tab |
| `Cmd+Shift+]` | Next tab |
| `Cmd+R` | Toggle auto-resume |

## Panes

| Shortcut | Action |
|----------|--------|
| `Cmd+D` | Split right |
| `Cmd+Shift+D` | Split down |

## Windows

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New window |
| `Cmd+Shift+N` | Duplicate window |

## Workspaces

| Shortcut | Action |
|----------|--------|
| `Cmd+Opt+N` | New workspace |
| `Cmd+Opt+Shift+N` | Duplicate workspace |

## General

| Shortcut | Action |
|----------|--------|
| `Cmd+O` | Open file |
| `Cmd+P` | Quick Open file search (or double-press `Opt`) |
| `Cmd+Shift+L` | Connect this agent to another ([Agent Bridge](/features/agent-bridge/)) |
| `Cmd+Shift+M` | Open the Mesh cockpit ([Mesh Workspace](/features/mesh-workspace/)) |
| `Cmd+Shift+C` | Toggle composer dock |
| `Cmd+E` | Toggle notes panel |
| `Cmd+B` | Toggle sidebar |
| `Cmd+F` | Find in terminal |
| `Cmd+K` | Clear terminal + scrollback |
| `Cmd+` `+` | Zoom in |
| `Cmd+` `-` | Zoom out |
| `Cmd+0` | Reset zoom |
| `Cmd+,` | Preferences |
| `Cmd+/` | Help |
| `Cmd+Q` | Quit |

## Editor

When an editor tab is active, these shortcuts override terminal shortcuts.

### Selection

| Shortcut | Action |
|----------|--------|
| `Cmd+D` | Select next occurrence |
| `Cmd+Shift+L` | Select all occurrences |

### Lines

| Shortcut | Action |
|----------|--------|
| `Opt+Up` | Move line up |
| `Opt+Down` | Move line down |
| `Shift+Opt+Up` | Copy line up |
| `Shift+Opt+Down` | Copy line down |
| `Cmd+Shift+K` | Delete line |

### Multi-Cursor

| Shortcut | Action |
|----------|--------|
| `Cmd+Opt+Up` | Add cursor above |
| `Cmd+Opt+Down` | Add cursor below |

### Editing

| Shortcut | Action |
|----------|--------|
| `Cmd+/` | Toggle line comment |
| `Tab` | Indent |
| `Shift+Tab` | Outdent |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |

### Search

| Shortcut | Action |
|----------|--------|
| `Cmd+F` | Find / replace |
| `Cmd+G` | Go to line (find next while the search panel is open) |
| `Cmd+Shift+G` | Find previous |

### File

| Shortcut | Action |
|----------|--------|
| `Cmd+S` | Save |

:::note
On Windows and Linux, replace `Cmd` with `Ctrl` and `Opt` with `Alt`.
:::
